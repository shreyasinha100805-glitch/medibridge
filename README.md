MediBridge 💊

A full-stack medicine reminder and caretaker monitoring system.

MediBridge helps patients remember medicines and allows caretakers to monitor whether medicines were taken on time.

🚀 Live Demo

Backend API

https://medibridge-backend-nq9w.onrender.com




🧠 Problem Statement

Many patients forget to take medicines on time.
Caretakers also struggle to track whether medicines were taken.

MediBridge solves this by:

Sending medicine reminders

Tracking medicine intake

Allowing caretakers to monitor patients
